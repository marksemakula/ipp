# International Education Pathway Program Website

A modern, corporate website built with Vue 3, TypeScript, and Vite.

## Features

- ✨ Modern and clean corporate design
- 📱 Fully responsive layout
- 🎨 Advanced CSS animations and transitions
- ⚡ Fast performance with Vite
- 🔒 TypeScript for type safety
- 🎯 SEO-friendly structure

## Tech Stack

- **Vue 3** - Progressive JavaScript framework
- **TypeScript** - Type-safe JavaScript
- **Vite** - Next generation frontend tooling
- **CSS3** - Modern styling with CSS Grid and Flexbox

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/       # Vue components
│   ├── Header.vue   # Navigation header
│   ├── Hero.vue     # Hero section
│   ├── About.vue    # About section
│   ├── Programs.vue # Programs showcase
│   ├── Benefits.vue # Benefits section
│   ├── Stats.vue    # Statistics section
│   ├── Testimonials.vue # Student testimonials
│   ├── Contact.vue  # Contact form
│   └── Footer.vue   # Footer section
├── App.vue          # Main app component
├── main.ts          # App entry point
└── style.css        # Global styles
```

## Sections

1. **Hero** - Engaging landing section with call-to-action
2. **About** - Program overview and key features
3. **Programs** - Detailed program offerings
4. **Benefits** - Why choose IPP
5. **Stats** - Key statistics and achievements
6. **Testimonials** - Student success stories
7. **Contact** - Application form and contact information
8. **Footer** - Links and additional information

## Customization

### Colors

Edit the CSS variables in `src/style.css`:

```css
:root {
  --primary-color: #c92a2a;
  --primary-dark: #a01f1f;
  --secondary-color: #fbbf24;
  /* ... */
}
```

### Logo

Replace the logo file in the `public/` directory.
